Welcome to Suprsearch!
A simple tool to perform multiple searches simultaneously.

Type your search term below, and add variables to the part of the search term that changes.


For instance, if you're interested in finding restaurants in different cities, use the following structure:


"Best restaurants in [city name]"


Replace [city name] in the variable column with the name of the city/cities you want to search for, such as "Brooklyn", "NYC" or "Portland".
